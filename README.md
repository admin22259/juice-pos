# نظام كاشير محل العصاير

هذا مشروع بسيط لنظام كاشير لمحل عصاير باستخدام HTML وCSS وJavaScript.

## طريقة الاستخدام
1. افتح index.html في المتصفح.
2. اختر المنتجات واضغط "الدفع".
3. السلة ستعرض المنتجات والإجمالي.

## المزايا
- إضافة منتجات للسلة.
- عرض إجمالي السعر.
- تفريغ السلة بعد الدفع.
